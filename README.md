# Loadforge Agent
* ⚠️ **Early Development**: This project is in early stages of development. 
The core functionality is not yet implemented.

LoadForge Agent is a lightweight, high-performance agent for distributed load testing. 
It executes test scenarios defined by LoadForge and reports metrics back to the control plane.

## Description
LoadForge Agent will be a lightweight, high-performance agent that executes test scenarios and
reports metrics back to the LoadForge control plane.

## Planned features:
* HTTP/HTTPS, WebSocket, and gRPC protocol support
* Real-time metrics collection and reporting
* Docker and Kubernetes deployment
* Distributed coordination for large-scale tests
